## 5. 功能层开关决策

默认全关。按业务模型逐个判定是否开启。

| 功能 | 默认 | 本客户是否开 | 理由 |
|---|---|---|---|
| productWizard（选型向导） | ❌ | | 需要 ≥30 SKU 且产品参数可多维筛选 |
| roiCalculator（ROI 计算器） | ❌ | | 需要单产品可量化回本，方案整合商慎用 |
| stepForm（3 步询盘表单） | ❌ | | 高客单价设备适用 |
| catalogDownload（PDF 门控） | ❌ | | 需要真实 PDF 目录素材 |
| caseStudies | ❌ | | 需要真实案例素材 |
| solutions | ❌ | | Solution Integrator 必开 |
| applications | ❌ | | Material Supplier / Component 厂适用 |
| productCompare | ❌ | | 产品间有明确对比维度时适用 |
| videoEmbed | ❌ | | 有真实产品视频才开 |
| exitIntent（退出弹窗） | ❌ | | **B2B 询盘站一般不开**（反感度 > 转化增益）|
| blog | ❌ | | 有真实内容来源才开（占位稿反而扣分）|
| programmaticSEO | ❌ | | **高风险**：需要真实差异化内容源，没有就是 thin content |
| aiAdvisor | ❌ | | 需要部署 Workers AI，第二阶段考虑 |
