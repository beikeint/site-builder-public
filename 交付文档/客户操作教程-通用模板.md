## ✅ 任务 1：开通询盘表单接收邮件（最重要，5分钟）

**为什么**：网站的"Contact"页面有联系表单，买家填完点提交后，邮件会发到你邮箱。现在表单还是占位符，**提交了收不到邮件**。

### 操作步骤

1. 浏览器打开 https://web3forms.com/
2. 点右上角 **Create Access Key**（免费）
3. 填一个邮箱 — **强烈建议用 `{{BUSINESS_EMAIL}}`**（这个邮箱才是询盘统一收件箱，别用私人邮箱）
4. 填完点 **Create Access Key**，立刻会收到一封确认邮件 → 打开邮件点里面的验证链接
5. 验证后网页会给你一串 Access Key，长这样 `abcd1234-5678-9abc-def0-1234567890ab`
6. **把这串 Access Key 发给我**（微信/邮件均可），我5分钟内把它嵌到网站里，表单立刻可用

### 怎么测试表单能收到邮件？

- 等我接上 key 后我会跟你说"可以测了"
- 你打开 {{SITE_URL}}/en/contact/ ，填个假名字假公司，消息写"test"，点 Send Inquiry
- 打开 {{BUSINESS_EMAIL}} 邮箱，应该收到一封主题 "{{BRAND}} Inquiry" 的邮件
- 没收到就告诉我，我查日志

> 📮 **信箱 {{BUSINESS_EMAIL}} 在哪？** 如果这个邮箱还没开通，去你的域名管理面板（Hostinger / cPanel）里创建一个邮件账户；或者你先用其他邮箱注册 key，等 {{BUSINESS_EMAIL}} 开通后我再换。

## ✅ 任务 2：开通 Google Analytics 4（流量统计，10分钟）

**为什么**：没有 GA4 你就看不到网站每天有多少人访问、来自哪些国家、看了哪些页、询盘转化率多少。这是运营的眼睛，**每个站独立建 Property**（不能和其他站混在一起）。

### 操作步骤

1. 浏览器打开 https://analytics.google.com/（用你常用的 Google 账号登录）
2. 左下角齿轮 → **管理(Admin)** → 左栏 **Create → Account** （如果已有 account 可跳）
3. Account 名字填：`{{BRAND}}`
4. 继续 → **Create Property**
5. Property 名字填：`{{BRAND}} Website`
6. 报告时区选 `(GMT+08:00) Shanghai`
7. 货币选 `Chinese Yuan (CNY)` 或 `US Dollar (USD)`都行
8. 商业目标里勾选：`Generate leads` + `Examine user behavior`
9. 填公司规模（10人以下）、行业（`{{INDUSTRY}}`）
10. 最后 → Data Stream 选 **Web** → Website URL 填 `{{SITE_URL}}`，Stream name 填 `{{BRAND}}`
11. 创建完会跳出 **Measurement ID**，长这样 `G-XXXXXXXXXX`
12. **把这串 G-XXXXXXXXXX 发给我**

### 建议顺便做的事

- GA4 里设置目标：Admin → Events → 会看到我们已经埋点的事件：`generate_lead` / `whatsapp_click` / `email_click` / `phone_click` / `form_start`。这5个都标记为 Key Event（原来叫 Conversion）
- 之后每周我会出 GA4 流量报告

## ✅ 任务 3：注册 Google Search Console（Google 收录，10分钟）

**为什么**：Google 爬虫需要知道你的新站存在。GSC 是 Google 官方的"爬虫控制台"，能看到：什么词在排名、哪些页被索引了、哪些有问题。

### 操作步骤

1. 浏览器打开 https://search.google.com/search-console
2. 用同一个 Google 账号登录（和 GA4 保持一致，数据能互通）
3. 点 **Add Property**
4. 选 **URL prefix** 方式，填 `{{SITE_URL}}`
5. 验证所有权：选 **HTML tag** 方式，复制给你一个 `<meta name="google-site-verification" content="XXXXXXXX">` 标签
6. **把这串 content 值 `XXXXXXXX` 发给我**，我嵌到网站 `<head>` 里
7. 嵌完后回到 GSC 点 **Verify** → 通过
8. 通过后到左栏 **Sitemaps**，提交 `{{SITE_URL}}/sitemap-index.xml`
9. 3-7 天后会看到第一批数据

## ✅ 任务 4：生成 IndexNow Key（加速收录，2分钟）

### 操作步骤（任选其一）

直接给我一句话："运营人员你帮我生成一个 IndexNow key"
我后台用脚本生成一串 32位字符的 key，你不用操作。

#### 方法 B（你自己来）
1. 打开 https://www.bing.com/indexnow
2. 点 Get your API key
3. 复制那串 key（类似 `YOUR_API_KEY_HERE`）
4. 发给我

## 常见问题 FAQ

**Q1：`{{BUSINESS_EMAIL}}` 邮箱没开通怎么办？**
去 Hostinger / cPanel 面板 → Emails → Create Email Account，选域名 {{DOMAIN}}，用户名填 info，密码自己设。开通免费。

**Q2：Google Analytics 建完看不到数据？**
正常，需要24小时开始有数据，7天才稳定。设置完告诉我 ID，我会去 Debug View 验证追踪是否正常工作。

**Q3：为什么 Web3Forms 不用付费版？**
免费版每月 250 条，够用。月询盘超 250 再升级，Pro 版 $12/月。

**Q4：能不能不做 GSC 只做 GA4？**
不能。GA4 告诉你"多少人来"，GSC 告诉你"为什么来"。两个都做才完整。新站不做 GSC 等于放弃 Google 自然流量。

**Q5：我发给你的素材版权问题？**
只用你自己拍/自己有版权的图。绝不会用竞争对手或供应链厂家的图，避免侵权。

**Q6：每个站都要独立建 GA4 Property 吗？**
是的，不能混在一起。混在一起后续没法分站做报告/做对比，价值大打折扣。
