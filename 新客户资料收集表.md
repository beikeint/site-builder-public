## 零、业务定位 5 问（🚨 v2.4 核心前置）

> 这 5 个问题决定网站的整体叙事方向。请用自己的话回答，不用套官话。

| # | 问题 | 你的回答 |
|---|------|---------|
| 1 | 公司收入 **80%** 来自"卖产品（单品/按 SKU）"还是"卖方案/整体服务"？ | |
| 2 | 产品是**自家工厂生产**的，还是**整合别家 + 自家**，还是**完全不生产只做贸易**？ | |
| 3 | 你有**自己的品牌**吗？还是主要为**别人品牌代工**？还是**没品牌，纯靠渠道赚差价**？ | |
| 4 | 你的**主要买家**是谁？ ① 终端消费者 ② 其他制造商（要用你的材料造东西）③ 工厂主（要建产线/扩产）④ 品牌商（让你代工）⑤ 分销商/进口商 ⑥ 需要服务的企业 | |
| 5 | 如果让你用**一句话**告诉陌生买家"为什么选你不选同行"，你会说什么？（不要说"高品质 / 专业团队 / 30 年经验"——这些所有同行都会说。要说具体的独特点） | |

**运营人员建站前必看**：根据以上回答判定客户属于 7 种业务模型哪一种：
- Pure Manufacturer（纯制造商）
- Solution Integrator（方案整合商）
- OEM/ODM（贴牌代工）
- Brand Owner（品牌商）
- Trader（贸易商）
- Material Supplier（材料供应商）
- Service Provider（服务商）

**判错此步 = 整站定位错**（2026-04-17 Demo-A 事件教训：Solution Integrator 被误判为 Pure Manufacturer，按产品+参数建站，客户反馈混乱）。
